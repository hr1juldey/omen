# Blender Addon Installation

To install Omen as a Blender addon (works alongside Mitsuba-Blender):

1. Create folder: `blender-addons/omen/`
2. Copy this entire project to that location
3. In Blender: Edit → Preferences → Add-ons → Install
4. Select this folder
5. Enable "Omen"

The addon will auto-register with Mitsuba 3 when enabled.
